# Major Minor | AI Financial Intelligence

## Project Overview
**Major Minor** is an advanced, AI-driven stock prediction and market analysis platform. It leverages machine learning architectures and real-time Natural Language Processing (NLP) to provide financial intelligence for global assets, including stocks, ETFs, and cryptocurrencies. 

Rather than relying purely on technical price history, the project integrates real-time **News Sentiment Analysis** to ensure that models correctly adapt to sudden geopolitical, breaking news, or crisis events (e.g., wars, pandemics) that would otherwise break traditional technical forecasting.

---

## Core Technologies
*   **Backend Engineering:** Python, FastAPI, Uvicorn, Pandas, Scikit-learn, TensorFlow/Keras.
*   **Data Pipelines:** `yfinance` API (for real-time historical quotes and breaking news aggregation).
*   **NLP Engine:** `vaderSentiment` (Rule-based sentiment classification).
*   **Frontend Technologies:** Pure HTML, Vanilla JavaScript, CSS3 Glassmorphism, Chart.js (Data Visualization).
*   **Resiliency:** Designed with "Graceful Degradation" so the project simulates organic market drift natively if heavy machine learning dependencies are locally unavailable.

---

## User Interface (UI) Architecture

The application is styled using an ultra-modern, dark-themed **Glassmorphism** aesthetic tailored for high-end fintech platforms. It utilizes glowing orbs, smooth grid transitions, and distinct status colorization.

### 1. Terminal View (Main Dashboard)
*   **Control Center:** The primary input grid where users can search for any global asset ticker. An async autocomplete dropdown connects to Yahoo Finance to suggest valid symbols.
*   **Engine Selection:** Users can choose which Neural Engine (e.g., LSTM, Random Forest) will process the data.
*   **Market Trajectory Chart:** An interactive, dynamic Chart.js canvas. It automatically plots the last 90 days of real closing prices (cyan line) alongside the 5-day AI forecasting projection (purple dashed line).
*   **Real-Time News Analysis:** A dynamic news feed generated by the NLP sentiment model. It displays the latest headlines, publishers, and auto-generated summaries. It calculates an "Overall Market Sentiment" badge (Bullish, Bearish, or Neutral) and highlights individual articles in Green, Red, or Grey based on their tone.

### 2. Models View
This section is an academic breakdown of the backend models currently plugged into the application logic, allowing users to understand the "explainability" of their AI:
*   **LSTM:** Explains how Deep Learning Recurrent Neural Networks handle volatile time-series clusters.
*   **Random Forest:** Details the tree-based ensemble method.
*   **Linear Regression:** Details the statistical baseline.

### 3. Documentation View
A systemic guide to the data pipeline, detailing how Scikit-learn normalizes large datasets into 0-1 metrics before gradient descent optimizations.

## How to Run the Project Manually

To boot up both the AI Backend API and the static web frontend cleanly, follow these exact steps:

### Step 1: Initialize the Environment
Open a Terminal/PowerShell window and navigate to the project directory:
```bash
cd "C:\Users\HARSHIL\Desktop\finance with ai TEST"
```

Activate the isolated Python virtual environment:
```bash
# On Windows PowerShell
.\venv\Scripts\Activate.ps1

# On Windows CMD
.\venv\Scripts\activate.bat
```

### Step 2: Install Dependencies
Ensure all the required machine learning modules and API packages are installed inside the virtual environment:
```bash
pip install -r requirements.txt
```

### Step 3: Start the Backend AI Core
Launch the FastAPI backend server using the `uvicorn` engine. This binds the API to port 8000:
```bash
python backend/main.py
```
*You should see a message stating `Application startup complete.`*

### Step 4: Access the Frontend
Because the frontend is built using standard static files, you do not *need* a frontend server. You can literally just open the file:

1. Open your File Explorer.
2. Navigate to `finance with ai TEST \ frontend`.
3. Double-click `index.html` to open it in Chrome, Edge, or Firefox.

*(Optional Alternative)*
If you prefer to serve the frontend via localhost, open a **separate concurrent terminal window**, navigate to the project, and run:
```bash
cd frontend
python -m http.server 8001
```
Then go to `http://localhost:8001` in your web browser.
